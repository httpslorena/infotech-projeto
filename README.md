## 📌 Sobre o Projeto

A **Infotech** é uma consultoria especializada em infraestrutura de dados e segurança da informação, focada em resolver gargalos de desorganização sistêmica e vulnerabilidades digitais em PMEs.

O ecossistema **Hexaguard** utiliza uma arquitetura híbrida para garantir:

- ✅ Resiliência operacional  
- ✅ Conformidade financeira  
- ✅ Segurança de dados  
- ✅ Automação de processos críticos  

---

## 🚀 Funcionalidades Principais

### 🖥️ Monitoramento Proativo

- Scripts Python para **Health Checks automatizados**
- Monitoramento contínuo de servidores críticos
- Alertas operacionais preventivos

### 👤 Gestão de Identidade (IAM)

- Implementação de **RBAC**
- Autenticação via **JWT**
- Uso obrigatório de **MFA**

### 💳 Segurança Financeira

- Adequação ao padrão internacional **PCI DSS v4.0**
- Proteção de dados financeiros sensíveis

### 📊 Business Intelligence

- Dashboard em tempo real
- Integração de KPIs financeiros e operacionais
- Visualização estratégica via **Power BI**

---

## 🛠️ Stack Tecnológica

- ☁️ **Infraestrutura:** Windows Server / AWS (RDS + S3)
- 🗄️ **Banco de Dados:** Microsoft SQL Server / PostgreSQL
- 🐍 **Linguagens:** Python (Pandas, PyODBC)
- 📊 **Visualização:** Power BI
- 📋 **Governança:** Scrum e ITIL v4

### 🔐 Destaque de Vulnerabilidade

A vulnerabilidade de **Controle de Acesso Quebrado** é mitigada através de:

- Validação centralizada no back-end
- Tokens assinados
- Políticas robustas de autenticação

## 📜 Adequação à LGPD

O sistema gerencia:

- 📂 25 tipos de processos de dados
- 📖 Fundamentação baseada no **Art. 7º da LGPD**
  - Execução de Contrato
  - Consentimento

### 🔐 Criptografia

- Dados sensíveis protegidos com **AES-256 em repouso**

### 🗑️ Direito ao Esquecimento

- Funcionalidade de **hard delete**
- Exclusão automática após 30 dias de carência

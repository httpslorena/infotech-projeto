1.1 Termo de Abertura (Project Charter)
EMPRESA
INFOTECH – Gestão Inteligente de Banco de Dados
PROBLEMA
Crescimento desordenado de dados dos clientes gera lentidão nas consultas, falhas de backup sem monitoramento centralizado e ausência de criptografia em bancos legados, expondo as empresas atendidas a riscos de vazamento e incompliance com a LGPD.
OBJETIVOS SMART
  • Reduzir o tempo médio de resposta de consultas SQL em 40% até o mês 4, via indexação e particionamento de tabelas.
  • Implantar rotina de backup automatizado com monitoramento centralizado em 100% dos clientes ativos até o mês 3.
  • Garantir criptografia AES-256 em todos os bancos com dados sensíveis até o mês 5, adequando-se à LGPD.

import pandas as pd

# Lendo os dados e garantindo que números sejam números
df = pd.read_csv("dados_projeto.csv", sep=";")

# Converte faturamento e horas para números (força erro a virar vazio)
df["Faturamento"]  = pd.to_numeric(df["Faturamento"],  errors="coerce")
df["Horas_Gastas"] = pd.to_numeric(df["Horas_Gastas"], errors="coerce")

# ETL: Remove quem não tem faturamento ou horas
df_limpo = df.dropna(subset=["Faturamento", "Horas_Gastas"]).copy()

# CÁLCULOS
faturamento_total = float(df_limpo["Faturamento"].sum())

total_projetos = len(df)
concluidos    = len(df[df["Status"].str.strip() == "Concluido"])
taxa_sucesso  = (concluidos / total_projetos) * 100

# KPI: Custo total (R$ 150/h)
custo_total = float((df_limpo["Horas_Gastas"] * 150).sum())

# RELATÓRIO FINAL
print("\n" + "="*40)
print("       RELATÓRIO FINAL - PROJETO GSC")
print("="*40)
print(f"Faturamento Total:   R$ {faturamento_total:.2f}")
print(f"Taxa de Sucesso:     {taxa_sucesso:.1f}%")
print(f"Custo de Operação:  R$ {custo_total:.2f}")
print("="*40)
print("STATUS: PARTE DE AUTOMAÇÃO CONCLUÍDA!")
print("="*40 + "\n")

except Exception as e:
    print(f"Erro ao processar: {e}")

3.2 Políticas de Acesso & 3.3 Adequação LGPD
// Políticas de Acesso
Autenticação Implementação de JWT (JSON Web Tokens) com expiração de 1 hora. Senhas armazenadas com bcrypt (hash + salt), nunca em texto plano.
Controle de Permissões (RBAC) Controle baseado em papéis: Admin tem acesso total ao sistema; Analista acessa apenas bases vinculadas ao seu contrato; Viewer tem acesso somente leitura.
Autenticação 2FA Duplo fator obrigatório para perfis Admin. Token TOTP via app autenticador (Google Authenticator / Authy).
Logs de Acesso Todos os acessos registrados com timestamp, IP e usuário. Retenção de 90 dias com alertas automáticos para acessos suspeitos.
// Adequação LGPD
Dados Coletados Nome, CPF, E-mail, Histórico de acessos e Logs de operações nos bancos de dados dos clientes.
Base Legal "Execução de Contrato" (Art. 7º, V) para dados operacionais e "Consentimento" (Art. 7º, I) para envio de comunicações e relatórios.
Criptografia Todos os dados sensíveis criptografados com AES-256 em repouso e TLS 1.3 em trânsito.
Direito ao Esquecimento Portal do titular no painel com opção "Excluir Conta e Dados", executando hard delete após 30 dias de carência, conforme Art. 18, VI da LGPD.

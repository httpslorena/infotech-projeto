3.2 Políticas de Acesso & 3.3 Adequação LGPD
// Políticas de Acesso
Autenticação Implementação de JWT (JSON Web Tokens) com expiração de 1 hora. Senhas armazenadas com bcrypt (hash + salt), nunca em texto plano.
Controle de Permissões (RBAC) Controle baseado em papéis: Admin tem acesso total ao sistema; Analista acessa apenas bases vinculadas ao seu contrato; Viewer tem acesso somente leitura.
Autenticação 2FA Duplo fator obrigatório para perfis Admin. Token TOTP via app autenticador (Google Authenticator / Authy).
Logs de Acesso Todos os acessos registrados com timestamp, IP e usuário. Retenção de 90 dias com alertas automáticos para acessos suspeitos.
// Adequação LGPD
Dados Coletados Nome, CPF, E-mail, Histórico de acessos e Logs de operações nos bancos de dados dos clientes.
Base Legal "Execução de Contrato" (Art. 7º, V) para dados operacionais e "Consentimento" (Art. 7º, I) para envio de comunicações e relatórios.
Criptografia Todos os dados sensíveis criptografados com AES-256 em repouso e TLS 1.3 em trânsito.
Direito ao Esquecimento Portal do titular no painel com opção "Excluir Conta e Dados", executando hard delete após 30 dias de carência, conforme Art. 18, VI da LGPD.
